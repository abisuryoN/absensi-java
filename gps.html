GPS diblokir Chrome (HTTP)

Chrome hanya mengizinkan GPS di HTTPS atau localhost.

Solusi 1 — Flag Chrome:

Buka tab baru, ketik:

chrome://flags/#unsafely-treat-insecure-origin-as-secure

Tambahkan:

Set Enabled → Relaunch


Solusi 2 — Gunakan tab "Input Manual" di atas untuk memasukkan koordinat secara manual.

Cara mendapatkan koordinat:
1. Buka Google Maps di HP/laptop
2. Tahan lama pada titik lokasi kantor
3. Koordinat muncul di bagian bawah
4. Copy ke kolom di bawah

Contoh: Latitude: -6.474124 Longitude: 106.748823
